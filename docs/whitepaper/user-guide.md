# User Guide

## Getting Started

### Installation

1. **Chrome Extension Installation**
   - Visit Chrome Web Store
   - Search for "MEMEFANS"
   - Click "Add to Chrome"
   - Complete installation

2. **Initial Setup**
   - Auto-generated SOL wallet
   - View wallet details
   - Backup private key
   - Basic settings

3. **Wallet Management**
   - **Built-in Wallet**
     * Automatic secure generation
     * View public address
     * Check balance
     * View transaction history
   
   - **Import/Export**
     * Export private key
     * Backup recovery phrase
     * Import existing wallet
     * Secure storage

### Basic Features

1. **Using on Twitter/X**
   ```
   Step 1: Visit Twitter/X
   Step 2: Find a tweet you want to interact with
   Step 3: Click extension icon
   Step 4: Create distribution for the tweet
   ```

2. **Distribution Settings**

3. **View Status**
